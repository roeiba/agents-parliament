# Aider

**Publisher:** Paul Gauthier (Open Source)

## Overview
Aider is a highly popular command-line AI pair programmer. It focuses on editing code in your local git repository.

## Key Features
- **Git Integration:** It is deeply integrated with Git; it commits changes with descriptive messages automatically.
- **Repo Map:** Uses a sophisticated "repository map" to understand the context of your entire project without filling the context window with unnecessary files.
- **Model Agnostic:** Works well with Claude 3.5 Sonnet, GPT-4o, and other top-tier models.
- **Diff Handling:** Excellent at applying diffs to files accurately, which is a common pain point for other agents.
