# Aider

**Publisher:** Paul Gauthier (Open Source)

## Overview
Aider is the premier AI pair programmer for Git-integrated code changes. It specializes in making clean, well-organized edits across multiple files with automatic commits.

## Primary Use Cases (When to Delegate to Aider)
- **Git-Integrated Changes**: Edits with automatic, well-described commits
- **Multi-File Refactoring**: Changes spanning multiple files while maintaining consistency
- **Repository-Wide Context**: Uses sophisticated "repo map" to understand project structure
- **Code Review & Fixes**: Applying review feedback across the codebase
- **Architect Mode**: High-level planning with two-model approach (plan + implement)

## When NOT to Use Aider
- Image generation or visual tasks (prefer Gemini)
- Web research or real-time information (prefer Gemini)
- Complex algorithmic reasoning (prefer Claude)
- Quick one-off snippets (prefer Codex)

## Key Features
- **Git Integration:** Automatic commits with descriptive messages
- **Repo Map:** Understands entire project context without filling the context window
- **Diff Handling:** Best-in-class at applying diffs accurately
- **Architect Mode:** Two-model approach for planning and implementation
- **Model Agnostic:** Works with Claude, GPT-4, and other models
