# GPT Engineer

**Publisher:** Anton Osika (Open Source)

## Overview
GPT Engineer differs slightly by focusing on "scaffolding" or building entire applications from a single prompt file.

## Key Features
- **One-Shot Generation:** specific focus on taking a `prompt` file and generating a complete codebase.
- **Clarification Loop:** Asks clarifying questions before writing code to ensure it understands the requirements.
- **Evolving:** Newer versions invoke agents to maintain and improve the code it generated.
