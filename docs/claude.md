# Claude Code

**Publisher:** Anthropic

## Overview
Claude is the most capable agent for complex coding tasks requiring deep reasoning, multi-step planning, and sophisticated problem-solving. It excels at architectural decisions and challenging algorithmic work.

## Primary Use Cases (When to Delegate to Claude)
- **Complex Coding**: Algorithmic challenges, system design, and architectural decisions
- **Deep Reasoning**: Multi-step problem solving and debugging
- **Long-Form Analysis**: Code reviews, security audits, and technical documentation
- **Structured Output**: JSON/schema-validated responses for programmatic use
- **Multi-Turn Conversations**: Extended context with skills and sub-agent spawning

## When NOT to Use Claude
- Image generation (prefer Gemini)
- Real-time web information (prefer Gemini)
- Simple one-off code snippets (prefer Codex)
- Git-integrated edits with auto-commits (prefer Aider)

## Key Features
- **Deep Reasoning:** Superior at complex logic and multi-step planning
- **200K+ Context:** Handles large codebases and lengthy conversations
- **Hooks:** Pre/post shell commands for workflow automation
- **Skills:** Loadable domain expertise (code review, security, etc.)
- **Sub-Agent Spawning:** Create specialized agents for sub-tasks
- **Claude Agent SDK:** Native Python integration without subprocess overhead
