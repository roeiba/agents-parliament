# Goose

**Publisher:** Block (Square)

## Overview
Goose is an open-source "on-machine" AI agent designed to automate engineering tasks.

## Key Features
- **Autonomous Operation:** Can plan and execute multi-step workflows like building a project, running tests, and debugging failures.
- **Local Focus:** Designed to run locally (privacy-conscious), effectively acting as a developer working on your machine.
- **Extensible:** Can be extended with "toolkits" to give it new capabilities.
