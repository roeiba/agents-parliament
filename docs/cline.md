# Cline (VS Code Extension)

**Publisher:** Open Source

## Overview
While primarily a VS Code extension, Cline is a significant agent in the ecosystem. It acts as an autonomous coding agent that can use your browser and terminal.

## Key Features
- **MCP Native:** Heavily built around the Model Context Protocol.
- **Human-in-the-Loop:** explicitly design to ask for permission before running commands or editing files (Safe-by-default).
- **Visual:** Being in VS Code, it provides a rich UI for the agent's "thought process."
