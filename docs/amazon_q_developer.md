# Amazon Q Developer CLI

**Publisher:** Amazon Web Services (AWS)

## Overview
Formerly known as CodeWhisperer, Amazon Q Developer includes a CLI component heavily optimized for AWS workflows and general development.

## Key Features
- **AWS Optimization:** deeply understands AWS services, CDK, and infrastructure-as-code.
- **Security:** Built with enterprise security in mind, including scanning for vulnerabilities and ensuring code doesn't violate IAM policies.
- **Auto-Completion:** Provides terminal auto-completion and natural language-to-command translation.
