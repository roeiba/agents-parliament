# Manus

**Publisher:** Manus

## Overview
Manus is a research-focused AI agent that excels at gathering and synthesizing information from multiple sources. It's particularly strong at competitor analysis, market research, and building small applications.

## Primary Use Cases (When to Delegate to Manus)
- **Research**: Deep dives into topics, synthesizing information from multiple sources
- **Competitor Analysis**: Gathering and comparing data across competitor websites
- **Multi-Page Data Collection**: Reading and extracting information from multiple web pages (not scraping)
- **Small App Building**: Creating simple applications, prototypes, and tools
- **Market Research**: Analyzing trends, gathering market intelligence

## When NOT to Use Manus
- Image generation (prefer Gemini)
- Complex algorithmic coding (prefer Claude)
- Git-integrated code changes (prefer Aider)
- Single-file focused edits (prefer Codex)
- CI/CD pipelines (prefer Goose)

## Key Features
- **Web Reading**: Can navigate and read multiple web pages to gather information
- **Information Synthesis**: Combines data from multiple sources into coherent analysis
- **Search Integration**: Built-in search capabilities for finding relevant information
- **Small App Development**: Quick prototyping and building of simple applications
- **Structured Output**: Can organize research findings into actionable reports
