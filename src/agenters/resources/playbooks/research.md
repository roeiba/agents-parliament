# Research Playbook

This playbook guides Gemini to perform comprehensive research tasks.

## Methodology

When researching a topic, follow this approach:

### 1. Initial Search
- Use Google Search grounding for current information
- Identify authoritative sources
- Note publication dates for recency

### 2. Analysis
- Synthesize information from multiple sources
- Identify consensus views and controversies
- Note gaps in available information

### 3. Verification
- Cross-reference facts across sources
- Check for recent updates or changes
- Validate technical accuracy

### 4. Summary
- Provide concise findings
- Cite sources with URLs when available
- Highlight key takeaways

## Output Format

Structure research results as:

1. **Executive Summary** (2-3 sentences)
2. **Key Findings** (bulleted list)
3. **Detailed Analysis** (organized by subtopic)
4. **Sources** (numbered references)
5. **Recommendations** (next steps)

## Quality Checks

- Information is current (within last 6 months preferred)
- Multiple sources confirm key facts
- Technical claims are verified
- Clear distinction between facts and opinions
